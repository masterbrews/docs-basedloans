# Technical

Based Loans is built using non-custodial smart contracts designed to enable fixed-term lending without liquidations.

All loan mechanics, collateral transfers, and lender settlements are handled on-chain, creating predictable outcomes for both borrowers and lenders.

This section provides access to the core technical infrastructure behind the protocol.

## Why It Matters

• Smart contracts enforce loan terms and collateral handling\
• No liquidation mechanisms or forced selling of collateral\
• Transparent on-chain infrastructure anyone can verify

Based Loans is designed to provide a clear and predictable lending system built on verifiable smart contract logic.
