# DAO & Governance

Based Loans is a Monstro Labs product, governed at the protocol level by Monstro DAO. These are two distinct entities with different roles: Labs builds and operates the protocol day-to-day, while the DAO owns the contracts and controls the parameters that matter most.

Neither entity can access user funds. Lender USDC and borrower collateral are controlled entirely by user actions and the protocol's smart contract logic.
