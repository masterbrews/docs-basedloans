# Community

Based Loans is built in public and supported by an active community of borrowers, lenders, and token holders. Whether you have a question about a loan, want to discuss a new token listing, or just want to follow along as the protocol grows, there are a few places to find us.

Updates and announcements go out on X first. Community discussion and support happen on Discord. Both are open to everyone.
